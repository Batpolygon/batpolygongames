PlushNToy est un projet de "moteur de jeu" développé sous Unreal Engine.Avec au moins un nouveau jeu publié chaque année, le projet évolue progressivement pour devenir de plus en plus complet et abouti au fil du temps. Le développement a commencé le 7 octobre 2022, avec l’idée initiale de créer un jeu de plateforme et d’énigmes où l’on incarne trois jouets : Man le mannequin, Pel la peluche et Pan le pantin. Cependant, la véritable motivation derrière ce projet est de recréer les aventures imaginaires que je partageais avec mes sœurs.

Mes derniers jeux sous PlushNToy:

Cloud Trials
Platroomer

Changelog :

Version Test 07012024 [07/01/2024] :
 La VT-07012024, développée sous Unreal Engine 4, est la version utilisée pour créer un deuxième jeu nommé : Cloud Trials.
Ce jeu reprend l’idée de base du projet : un jeu de plateforme et d’énigmes où l’on incarne trois jouets principaux : Man le mannequin, Pel la peluche et Pan le pantin.
Cette version améliore de nombreux aspects du moteur, comme le développement d’un HUB central avec des niveaux, un mode coopération, et un level design plus abouti.
Elle marque aussi un grand pas dans la construction de "notre" univers, inspiré des aventures que je partageais avec mes sœurs.
L’histoire de Cloud Trials se déroule dans une partie spécifique de cet univers : le monde des nuages.

Version Test 27012023 [07/01/2023] :
 La VT-27012023, développée sous Unreal Engine 4, est la toute première version utilisée pour créer un jeu : Platroomer.
Ce jeu a permis de poser les bases du moteur en explorant le gameplay orienté plateforme, l’intégration de pouvoirs, et des systèmes comme le HUD et la génération aléatoire de parcours et de jouets.
L’idée de Platroomer vient d’un jeu que je faisais avec mes deux sœurs quand nous étions enfants : on utilisait des objets de notre chambre pour créer des parcours et jouer avec nos jouets.
D’ailleurs, la chambre modélisée dans le jeu est inspirée de celle où nous jouions ensemble.
